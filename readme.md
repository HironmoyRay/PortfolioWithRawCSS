## Features

- **Responsive Design**: Ensures that the site works well on various screen sizes.
- **Navigation Bar**: Includes links to Home, About, Portfolio, Contact, and Blog Site.
- **Hero Section**: Split into two parts - a description section about me and an image.
- **Reusable Data**: The site maintains a similar structure and look across different pages, while displaying different content.
- **Portfolio Section**: Highlights key projects, each with consistent styling and presentation.
- **Blog Section**: Lists recent blog posts. Update available soon...
- **Font Awesome Icons**: Icons used across the website for better visual representation.

## Technologies Used

- **HTML5**
- **SCSS**: For styling and organizing CSS with variables and mixins.
- **CSS Flexbox & Grid**: For layout management.
- **Font Awesome**: For icons.
- **CSS Gradient Magic**: Utilized for creating color gradients.
